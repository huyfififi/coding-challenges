# Step 1

## `int()`の第二引数、`bin()`を利用する方法

`int()`の第二引数に基数を指定できることを知っていたので、それを利用する方法が最初に思いついた。

```
In [1]: int("10")
Out[1]: 10

In [2]: int("10", 10)
Out[2]: 10

In [3]: int("10", 2)
Out[3]: 2
```

```
In [1]: bin(2)
Out[1]: '0b10'
```

## 数学で習った二進数の足し算を愚直に実装する方法

数学で習った通り、二進数（のみならず他の基数にも対応）の足し算を実装。実装自体はすんなりできたが、`answer`・`digit`あたりの変数名で良い案が思いつかなかった。他の方々の命名を眺めてみることにする。
